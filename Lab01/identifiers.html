1. Atomic number

  • The atomic number or proton number (symbol Z) of a chemical element is the number of protons found in the nucleus of every atom of that element.

  • Example: Atomic number of Oxygen is 8

2. City/province identifier of the citizen identification number

  • The first 3 digits of the citizen's identification number identify the province in which the citizen was born.

  • Example: City identifier of Hanoi is 01

3. Class code

  • The course code is the identifier for a course of a school.

  • Example: IT4552E is course code of Web Programmming

4. Student Number

  • Student Number is the identifier for a student of a school.

  • Example: My Student number is 20194728

5. Health insurance card number

  • Health insurance card offers cashless payment options to pay for your medical bills arising from hospitalisation and other treatment charges.

  • Example: CN 3 01 0003500099
